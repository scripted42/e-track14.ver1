## 🔧 Quick Setup Instructions

### 1. Clone the Repository
```bash
git clone https://github.com/scripted42/e-track14.ver1.git
cd e-track14.ver1
```

### 2. Backend Setup
```bash
cd backend
composer install
cp .env.example .env
php artisan key:generate
php artisan migrate
php artisan serve
```

### 3. Mobile App Setup
```bash
cd ../frontend-mobile
flutter pub get
flutter run
```
